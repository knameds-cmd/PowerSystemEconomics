# Data Specifications

Store data dictionaries, source catalogs, and input checklists here.

The current workbook in this folder describes the expected raw files, naming rules, and how they connect to the Julia data-loading pipeline.

Additional repo-ready references:

- `api_source_catalog.md`: cleaned summary of approved APIs, endpoints, and field usage.
- `api_collection_plan.md`: how to collect and stage the raw datasets safely.
- `api-guides/`: original provider guide documents copied into the repository for reference.
- `../../config/data_sources.template.toml`: machine-readable template for future loaders.
