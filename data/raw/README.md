# Raw Data

Store original API responses and downloaded source files here.

Suggested subfolders:

- `smp_forecast_demand/`
- `renewables/`
- `power_market_gen_info/`
- `fuel_cost/`
- `smp_decision_by_fuel/`
- `reference/`

Prefer saving the original response first, then transforming it into cleaned tables under `data/processed/`.
